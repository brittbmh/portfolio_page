Work

Bonne Fête

This app was my first Capstone Project at Prime. We were given two weeks to create a solo project. I created a party planning app that allows hosts to both provide details to their guests and to get the information they want back from the guests. It allows the host to select the questions that they want to ask, and then display the responses to those questions in a table. Guests are also able to log in to the app to track their invites and responses.

Tech Used: React.js, Node.js/Express.js, AJAX/Axios, Redux.js, Redux-Saga, PostgreSQL, Sweetalert.js, HTML, CSS
